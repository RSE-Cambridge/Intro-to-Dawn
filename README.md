## About The Project

Training resources for an introduction to the Dawn HPC system for beginners. This covers:

1. How can High Performance Computing benefit you?
2. Connecting to Dawn
3. System Architecture
4. Running jobs using SLURM
5. Accessing Software Applications
6. Transferring files to/from Dawn
7. Final Task: Running a job with Multi-GPUs

## Getting Started

To take this course, you will need a registered account on Dawn. Details for requesting access can be found [here](https://).

### Prerequisites

This course is for complete beginners, however some familiarity with the following may be beneficial:

- Bash shell
- Python
